关于物品名字本地化
  
  * 原版物品
  
  > 物品名字读取于插件中配戴的ItemName_xx_XX.lang<br />
  > 如果对应的翻译文件不存在,会自动从指定的连接下载相应的资源
  
  * 物品
  
  > 物品名字读取自Forge的API类cpw.mods.fml.common.registry.LanguageRegistry<br />
  > 如果mod中不存在对应的语言文件,可以把翻译文件拖到mod文件的语言文件夹
